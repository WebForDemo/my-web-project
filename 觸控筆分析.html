Core Technologies

四大觸控筆技術一覽

市面上的觸控筆主要有四種技術原理。每一種在「精準度」、「是否需要充電」以及「設備相容性」上都有根本性的差異。

🧲

電磁筆 EMR

Electromagnetic Resonance

由 Wacom 主導的技術。螢幕底層貼有電磁感應層 (Digitizer)，持續向外發射電磁波。 筆內部的線圈被動接收能量後，將壓力等資訊反射回螢幕。
筆完全不需要電池，是最接近「紙筆」的書寫體驗。

✓ 不需充電 ✓ 壓感精細 ✓ 懸浮偵測 ✓ 防誤觸 ✗ 需專用設備
⚡

主動式電容筆

Active Electrostatic (AES / MPP)

筆身內建電池與微型晶片，由筆主動發出電訊號至電容螢幕。 可做到極細筆尖（~1.5mm）、壓力感應、傾斜偵測等進階功能。 通常搭配藍牙進行壓感/手勢資料傳輸。
沒電就完全無法使用。

✓ 壓感 + 傾斜 ✓ 筆尖極細 ✓ 藍牙快捷鍵 ✗ 必須充電 ✗ 限指定設備
👆

被動式電容筆

Passive Capacitive Stylus

模擬手指靜電，靠筆頭的導電材質將人體電荷傳導至螢幕。不需要電池， 但筆尖必須做得粗圓（橡膠頭或圓盤頭）才能產生足夠的接觸面積。
所有觸控螢幕都能用，但沒有壓感、沒有防誤觸。

✓ 不需充電 ✓ 通用性極高 ○ 成本最低 ✗ 無壓感 ✗ 無防誤觸 ✗ 精準度差
🔴

電阻式觸控

Resistive Touch

螢幕有兩層導電層，靠物理壓力使上下層接觸通電來判斷位置。 任何硬物都能操作（手指、指甲、原子筆頭、手套）。常見於停車場繳費機、 提款機等公共設備。
耐操、抗水滴干擾、不挑操作工具。

✓ 任何物體可操作 ✓ 防水滴干擾 ✓ 成本低廉 ✗ 需施力按壓 ✗ 不支援多點觸控 ✗ 透光度較差

💡 用水池來比喻四種技術

把螢幕想像成一片水池，每種筆就是不同的「擾動方式」——

  • 電阻式：用力戳水面讓兩層水接觸 → 需要物理壓力，什麼東西都能戳
  • 被動電容：用帶電的手指「碰觸」水面引起波紋 → 只有導電體有效
  • 主動電容：自帶一台馬達往水裡打水花 → 訊號強、準、但需要供電
  • 電磁 (EMR)：水池底部有磁鐵在轉，筆裡的金屬片被吸引會震動 → 完全靠水池供能
Active vs Passive

主動式 vs 被動式：關鍵在「電力來源」

判斷一支筆是「主動」還是「被動」，最根本的標準只有一個：筆本身是否需要自行供電來產生訊號。

🔋 被動式 Passive

筆身不需要電池，不具備發射訊號的能力。它只是在「干擾」或「反應」螢幕原有的場域。

代表：被動電容筆（圓頭）、EMR 電磁筆
VS

⚡ 主動式 Active

筆身內建電路與電池，會「主動」產生能量來發出訊號，告訴螢幕位置與壓力。

代表：Apple Pencil、Surface Pen、平價副廠筆
⚠️
常見混淆：EMR 電磁筆明明也很精準，為什麼是「被動式」？

因為 EMR 筆本身不發射訊號。它裡面的線圈只是「接收」螢幕 Digitizer 發出的電磁場能量， 再利用這個能量「反射」資料回去。就像一面鏡子——它不發光，但能反射陽光。
而主動式電容筆則像一支手電筒——自己裝電池發光，不論環境如何都能射出精準的光。

🔌 三種「需要電」的筆，差別在哪？

🧲 EMR 電磁筆 (有藍牙版)

如 Samsung S Pen（新款），寫字仍不需要電，電池只供給藍牙遙控功能（手勢操作、遙控拍照）。即使筆沒電，書寫功能完全不受影響。

✏️ 簡易主動筆（副廠）

要充電但不連藍牙。開機即寫，只是用電力讓細筆尖產生足夠電訊號。沒有壓感、沒有快捷鍵。適合預算有限的筆記需求。

🖊️ 全功能主動筆

要充電 + 連藍牙。全方位功能：壓感、傾斜偵測、快捷鍵、電量顯示。如 Apple Pencil、Surface Pen。沒電便無法使用。

一句話理解藍牙：

藍牙負責「附帶功能」——按鈕翻頁、切換工具、電量顯示、手勢遙控。

Hardware Layer

EMR 的核心硬體：Digitizer 數位感應層

EMR 設備的關鍵在於螢幕底下藏有一層專用的電磁感應電路板 (Digitizer)。 這層感應層在製造時就必須精準校準，無法事後外掛或貼上。

🔧 EMR 設備的「三明治結構」

🛡️ 保護玻璃 / 塑膠膜 手寫接觸面（可加貼類紙膜增加摩擦感）
📺 顯示面板 (LCD / OLED / E-Ink) 負責畫面呈現
👆 電容感應層 負責手指觸控感應（滑動、縮放）
🧲 EMR 電磁感應層 (Digitizer) 密密麻麻的感應線圈 — 負責偵測電磁筆位置與壓力

🔍 為什麼放在最底層？

EMR 透過磁場運作，磁場可穿透上方的玻璃、顯示面板等物理層。 放在底層不會擋住光線，螢幕看起來更通透，書寫體驗更自然。

❌ 可以自己「貼」一層嗎？

不行。Digitizer 在製造時必須與主機板晶片精準校準、佈線連接。 它不是像保護貼可以事後加裝。這也是 EMR 設備定價較高的原因之一。

🎯
關鍵結論：選筆之前，先選「設備」

與其在意筆的品牌，不如先確認你的平板螢幕裡有沒有 Digitizer：
— 有 Digitizer → EMR 陣營（筆不用電，如 MobiScribe、Boox、Samsung）
— 沒有 Digitizer → 只能用主動式電容筆（筆要充電，如 Apple Pencil）

Quick Identification

實戰辨識：四步驟判斷你的筆是哪一種

不需要拆開筆或看規格書，只要做幾個簡單測試就能分辨。

1
懸浮測試

筆靠近螢幕 0.5~1cm 但不碰觸，若螢幕出現游標 → 電磁筆 EMR。 這是最準確的判斷法。

2
跨裝置測試

拿筆去畫手機螢幕。有反應 → 電容筆（被動或主動）。 完全沒反應 → 電磁筆（手機沒 Digitizer）。

3
斷電測試

把筆放著一年不充電。拿起來還能寫 → EMR 或被動電容。 完全沒反應 → 主動式電容筆。

4
筆尖外觀

圓頭 / 圓盤 → 被動電容筆。 細尖 + 重量輕 → EMR 電磁筆。 細尖 + 明顯較重 → 主動式（有電池）。

🧪 終極辨識口訣

  • 能用指甲按的螢幕 → 電阻式（像繳費機、早期 GPS）
  • 筆浮空就有游標 → 電磁筆 EMR
  • 不充電 + 粗圓頭 + 什麼螢幕都能畫 → 被動電容筆
  • 要充電 + 細筆尖 + 只能畫特定平板 → 主動式電容筆

總結：選「設備」比選「筆」更重要

設備決定了你能用哪種技術的筆。先確認設備的螢幕規格（有沒有 EMR Digitizer）， 再挑選最適合的筆。

如果是 EMR 陣營，你甚至可以跨品牌選購 Wacom 相容筆（如 Lamy、Staedtler 施德樓出品的電磁筆）， 在 MobiScribe、Boox、Samsung 之間通用，完全不用充電！
